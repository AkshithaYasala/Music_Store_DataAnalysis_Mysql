# Music Store Data Analysis using SQL

## Project Overview

The Music Store Data Analysis project focuses on analyzing a digital music store database to uncover customer purchasing behavior, sales trends, and business performance. Using SQL, the project extracts meaningful insights from transactional data to support data-driven decision-making.


## Business Problem

The music store wants to understand customer behavior, identify top-performing artists and genres, analyze sales trends, and improve revenue generation through data-driven insights.


##  Project Objectives

* Analyze customer purchasing behavior.
* Identify top-selling artists, albums, and genres.
* Determine high-value customers and revenue contributors.
* Evaluate country-wise and city-wise sales performance.
* Generate actionable business insights using SQL.


##  Tools & Technologies

* **Database:** MySQL
* **Language:** SQL
* **Techniques:** Joins, Subqueries, CTEs, Window Functions, Aggregate Functions
* **Tool:** MySQL Workbench


## Database Tables

The database consists of multiple interconnected tables:

* Employee
* Customer
* Invoice
* Invoice_Line
* Track
* Album
* Artist
* Genre
* Playlist
* Media_Type


##  Key Business Questions Solved

1. Identify the senior-most employee.
2. Determine the countries with the highest number of invoices.
3. Find the top invoice values.
4. Identify the city generating maximum revenue.
5. Determine the best customer based on spending.
6. Find customers who listen to Rock music.
7. Identify top Rock artists.
8. Analyze customer spending on artists.
9. Determine the most popular genre by country.
10. Find the highest-spending customer in each country.


##  SQL Concepts Used

* INNER JOIN
* LEFT JOIN
* GROUP BY
* ORDER BY
* HAVING
* Subqueries
* Common Table Expressions (CTEs)
* Window Functions
* Aggregate Functions
* Ranking Functions


##  Key Insights

* Rock music emerged as one of the most popular genres.
* Certain customers contributed significantly to total revenue.
* The USA generated the highest number of purchases.
* A small number of artists accounted for a major portion of sales.
* Customer spending patterns varied across countries.
* Genre preferences differed by region.


##  Business Impact

The analysis helps the music store:

* Understand customer preferences.
* Improve marketing strategies.
* Promote popular artists and genres.
* Increase customer retention.
* Optimize product recommendations.
* Support revenue growth through data-driven decisions.


##  Conclusion

This project demonstrates how SQL can be used to transform raw transactional data into valuable business insights. By applying advanced SQL concepts, the project provides meaningful recommendations that support strategic decision-making and improve overall business performance.
